FAQ (アコーディオン)

<details> ベースなので JS 不要です。

yoru.css とはなんですか？
ダーク・テック方向に振り切った、1 ファイル完結の CSS パッケージです。LP・コーポレートサイト・ポートフォリオ制作を想定しています。
ビルドツールは必要ですか？
不要です。<link rel="stylesheet" href="yoru.css"> の 1 行だけで全スタイルが適用されます。
カスタマイズはできますか？
CSS 変数（カスタムプロパティ）でカラー・スペーシング・シャドウなどを一元管理しています。:root を上書きすれば全体のトーンを変更できます。
ライトテーマには対応していますか？
しません。yoru.css は「夜」に特化した設計です。ただし <body class="theme-soft"> でやや明るめのダークに切り替えられます。
